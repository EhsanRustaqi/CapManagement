# \# FleetManagement 

# 

# FleetManagement is a SaaS platform designed for fleet companies that rent qualified taxi vehicles to drivers working on platforms such as Uber, Bolt, and other ride-hailing services.

# 

# The system helps fleet owners manage vehicles, drivers, expenses, and financial settlements while ensuring accurate VAT (BTW) and cost calculations.

#
